# Browser 250: America's Sovereign Window 📜

Browser 250 is a self-contained, high-security web browser built to defend your digital sovereignty. By deploying an isolated browser engine directly inside a secure Docker container, Browser 250 completely masks your real device metrics, blocks telemetry/trackers, and safely bypasses aggressive anti-bot detection walls like Cloudflare Turnstile.

---

## 🚀 Two-Step Installation Guide

You do **not** need any command-line experience to run this application. Follow the simple steps below for your operating system:

### 🪟 Windows Setup (Windows 10 / 11)
1. Download this project folder to your computer and open it.
2. Right-click **`click_to_install.bat`** and select **Run as Administrator**.
3. *That's it!* The installer will check for your prerequisites, build the secure environment, and drop a clickable **Browser 250** launch icon straight onto your Desktop.

### 🍏 macOS Setup (Intel / Apple Silicon M1/M2/M3)
1. Download this project folder to your computer and open it.
2. Double-click the file named **`Setup_Mac.command`**.
3. *That's it!* A terminal window will open to safely configure your background engine. Once finished, a clickable **Browser_250** launch icon will appear straight on your Desktop workspace.

---

## 🛠️ How to Use Browser 250

Once the installation is complete, you never have to run setup again. 

1. Double-click the **Browser 250** icon on your Desktop.
2. Your computer's standard web browser will open and instantly display your secure, containerized desktop window at:
   `http://localhost:8080/vnc.html`
3. Click the **Connect** button on the screen to view your sovereign browsing space.
4. Click the **📜 Read the Sovereign Declaration of Digital Independence** link at the bottom of the home search page to view your browser's operational constitution.
